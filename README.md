# 📈 Real-Time Stock Market Dashboard

An interactive, real-time financial tracking dashboard built with Python, utilizing Streamlit for the user interface, Plotly for dynamic candlestick charts, and the yfinance API for live market data streams.

## ✨ Features
* **Live Price Tracking:** Real-time updates with customizable charting windows (1-day, 5-day, and 1-month).
* **Interactive Candlestick Visuals:** Powered by Plotly with hover metrics and auto-scaling.
* **Financial Performance Ratios:** Dynamic metric cards showing price variations, daily highs, and real-time trading volumes.
* **Rate-Limit Bypass:** Configured with robust browser user-agent sessions to ensure steady API data stream connection.
